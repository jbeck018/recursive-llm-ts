How It Works

Based on the 2025 MIT paper "Recursive Language Models" by Alex Zhang and Omar Khattab, RLM solves the fundamental context length limitation of LLMs.

Instead of truncating or failing on large documents, RLM recursively decomposes the problem:

  • Large context is split into manageable chunks
  • LLM writes JavaScript code to analyze each chunk
  • Code is executed in a sandboxed REPL (goja engine)
  • Results are recursively combined up the tree
  • For structured outputs, schemas are decomposed into subtasks
  • Parallel goroutines process independent fields simultaneously
  • Validation with retry ensures schema compliance

Quick Start

Get up and running in under a minute.

TypeScript 
import { RLM } from 'recursive-llm-ts';

const rlm = new RLM('gpt-4o-mini', {
  api_key: process.env.OPENAI_API_KEY,
  max_iterations: 15,
});

// Process a 100K+ token document
const result = await rlm.completion(
  'What are the key findings in this research paper?',
  longResearchPaper
);

console.log(result.result);
console.log('Stats:', result.stats);
TypeScript 
import { RLM } from 'recursive-llm-ts';
import { z } from 'zod';

const rlm = new RLM('gpt-4o-mini', {
  api_key: process.env.OPENAI_API_KEY,
});

const schema = z.object({
  sentiment: z.number().min(1).max(5),
  summary: z.string(),
  topics: z.array(z.enum(['pricing', 'features', 'support'])),
  keyPhrases: z.array(z.object({
    phrase: z.string(),
    weight: z.number(),
  })),
});

const result = await rlm.structuredCompletion(
  'Analyze the sentiment and extract key information',
  callTranscript,
  schema,
  { parallelExecution: true }
);

// result.result is fully typed!
console.log(result.result.sentiment);     // number
console.log(result.result.keyPhrases);    // Array<{phrase, weight}>

Go 
package main

import (
    "fmt"
    "os"

    "github.com/jbeck018/recursive-llm-ts/go/rlm"
)

func main() {
    engine := rlm.New("gpt-4o-mini", rlm.Config{
        MaxDepth:      5,
        MaxIterations: 30,
        APIKey:        os.Getenv("OPENAI_API_KEY"),
        MetaAgent: &rlm.MetaAgentConfig{
            Enabled: true,
        },
        Observability: &rlm.ObservabilityConfig{
            Debug: true,
        },
    })

    answer, stats, err := engine.Completion(
        "Summarize this document",
        largeDocument,
    )
    if err != nil {
        fmt.Fprintf(os.Stderr, "Error: %v\n", err)
        os.Exit(1)
    }

    fmt.Printf("Answer: %s\n", answer)
    fmt.Printf("LLM calls: %d\n", stats.LlmCalls)
}
